- Dashboard 定位为 `AI 可见度驾驶舱`，不是普通任务看板。
- 本地 smoke 数据和测试示例必须明确标注，不能包装成正式线上结果。
- 首屏优先回答“有没有推荐、哪些模型没推荐、为什么没推荐、下一步补什么”。
- 图表优先使用现有前端能力：轻量 SVG、CSS 条形图、标签组，不新增图表依赖。
- 只做页面表达和低风险前端映射，不改接口、不改数据库、不做自动诊断。
- 最终采用“指标卡 + 趋势图 + 模型对比 + 原因分布 + 标签云 + 行动清单”的企业级 SaaS 分析驾驶舱结构。

- 标题收口为 `AI 可见度驾驶舱`。
- 明确提示当前为本地测试版 / smoke 数据。

### 核心指标卡

首屏指标扩展为 6 个 GEO 方向指标：

- 品牌推荐率
- 品牌提及率
- 官网引用率
- 未命中率
- 竞品占位率
- 待补救问题
