- 发布文章的发布状态、资料依据、FAQ 和质量检查结果。

## 4. 轻量关联规则

- 问法与知识库：基于问法文本、产品线、场景词和资料标题 / 文件名 / 资料主题 / 来源说明做关键词弱关联。
- 问法与文章：优先使用 `geoPromptId` 直接关联；没有直接关联时用标题、正文、发布包摘要和关键词做弱关联。
- 问法与模型覆盖记录：优先使用 `geoPromptId` 直接关联；没有直接关联时用问法、摘要、原始回答和模型信息做弱关联。
- 证据状态：有可引用知识库资料时显示 `可引用`；有资料但未通过可引用规则时显示 `需标注`；没有弱关联资料时显示 `需补证据`。
- 文章状态：有发布就绪且包含资料依据的文章时显示 `引用友好`；有文章但依据或结构不完整时显示 `建议补充 / 需人工确认`；无文章时显示 `暂无文章`。
- 模型覆盖状态：优先看现有记录里的品牌推荐、品牌提及和未推荐状态，不从页面文案反推真实推荐。
- 可能引用来源：只展示 `官网 / 产品页`、`知识库`、`发布文章`、`发布内容资产` 或 `暂无明确来源`，并明确说明不代表 AI 已真实引用。

所有结果均为前端展示级判断，需要结合原始回答、资料来源和人工判断确认。
