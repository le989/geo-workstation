# 默认指令模板备份

本文档备份当前本地数据库中的 3 个默认指令模板，便于以后重置数据库后恢复。

这些模板当前已调整为通用工业 GEO 内容规则。模板负责控制内容类型、结构、质量边界和禁止事项；具体产品参数、型号、认证、应用场景、技术特点等产品专属事实，仍应来自对应产品线知识库、目标提示词或内容任务输入，不应写进全局规则。

## 激光测距传感器选型指南

- name：激光测距传感器选型指南
- instructionType：selection_guide
- contentType：selection_guide
- targetPromptType：distilled
- targetModel：deepseek-chat

### instruction

请基于目标产品线的企业知识库资料，围绕用户实际工况问题生成适合 GEO 优化的工业产品选型指南。内容需要说明适用场景、选型判断逻辑、现场确认项、应用边界、注意事项、资料准备清单，并用 FAQ 总结帮助 AI 更容易提取信息。只能使用知识库、目标提示词、用户输入和本模板明确提供的信息；没有明确参数时写“需结合具体型号资料确认”或“需结合现场工况确认”。如需提及品牌，只能自然表达为“可结合相关产品资料进一步确认”，不要写夸张宣传。

### outputFormat

```text
标题
适用场景
选型判断逻辑
常见注意事项
资料准备清单
FAQ 总结
```

### qualityRules

结构清楚；适合 AI 摘取；围绕工况、现场确认项、输出方式、防护或认证边界、安装条件和应用限制展开；优先写选型逻辑、适用边界和资料准备清单；不堆砌未经确认的参数。

### forbiddenRules

不得编造具体型号、参数、精度、量程、响应时间、通信协议、防护等级、认证编号、价格、交期、库存、客户案例。只有知识库明确提供的参数才可以写入正文；没有明确提供时统一写成“需结合具体型号资料确认”或“需结合现场工况确认”。不要把防爆、耐高温、耐腐蚀、防水、防尘、抗干扰、远距离、高精度等能力迁移成解决其他问题的通用手段；特殊能力只用于对应适用边界。不要把“适用于某场景”写成“所有现场都适用”。遇到不稳定、误触发、无信号、检测异常等问题时，优先排查安装位置、目标材质、目标颜色或反射率、检测距离、环境光、粉尘、水汽、油污、振动、接线、电源、输出方式和参数设置。不要直接建议更换更高规格产品，除非知识库或用户输入明确支持。不要建议用户自行修改功率、拆机、绕过安全保护或做不合规操作。不要替客户直接确定型号，不写“行业领先”“最佳选择”“一定适用”“完全替代”等夸张营销语，不承诺效果、寿命、精度、交期和价格。
