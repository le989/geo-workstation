# AI 拓词质量规则

本文档定义 GEO 工作站中 AI 拓词的质量边界。AI 拓词的目标不是生成普通 SEO 关键词，而是生成“真实用户可能会向 AI 提出的问题候选”。候选词只进入拓词任务结果，必须由人工勾选后才能保存到提示词策略库。

## 1. AI 拓词目标

- 生成自然语言问题或明确搜索意图。
- 覆盖用户在咨询、购买、报名、到店、合作或使用前后的关键决策场景。
- 服务后续 GEO 内容生成、模型覆盖记录和优化复盘。
- 避免关键词堆砌、广告口号和未经证实的具体承诺。

## 2. 高质量 GEO 提示词

高质量候选词通常具备以下特征：

- 像用户会直接问 AI 的句子。
- 包含明确场景、顾虑、比较对象、准备动作或判断逻辑。
- 能指导后续内容生产，例如指南、FAQ、场景方案、对比与替代、信任证明。
- 不依赖虚构参数、型号、价格、认证、案例或效果承诺。

示例：

- 工业现场远距离检测该怎么选传感器。
- 家长给孩子选一对一辅导前要问哪些问题。
- 公司团建聚餐选餐厅要看哪些条件。
- 本地生活服务商靠不靠谱应该怎么判断。

## 3. 低质量候选词

以下候选词应过滤或降低优先级：

- 空文本或过短文本。
- 与训练词完全相同的文本。
- 只有“产品词 + 厂家 / 价格 / 品牌 / 官网”的泛词。
- 明显重复的候选词。
- 包含虚构具体参数、认证编号、价格、承诺或效果保证。
- 包含“保证”“一定”“最好”“第一”“100%”等过度承诺表达。

示例：

- 激光测距传感器厂家。
- 光电传感器哪个好。
- 课程推荐。
- 服务多少钱。

## 4. 用户决策场景分类

AI 拓词按用户决策场景组织，而不是按固定行业组织。

| 场景       | 用户问题特征                         | userIntent 建议                                      |
| ---------- | ------------------------------------ | ---------------------------------------------------- |
| 需求决策   | 不知道怎么选、怎么判断、怎么开始     | `selection` / `purchase`                             |
| 问题诊断   | 遇到问题，需要判断原因和改善路径     | `troubleshooting`                                    |
| 场景方案   | 有具体场景，需要知道适合什么方案     | `application_solution`                               |
| 对比替代   | 在两个选择之间犹豫                   | `comparison` / `domestic_alternative`                |
| 信任验证   | 判断品牌、机构、门店或服务方是否可信 | `brand_verification` / `manufacturer_recommendation` |
| 行动前准备 | 咨询、购买、报名、到店或合作前准备   | `purchase` / `selection`                             |

## 5. 优先级判断

- P1：高价值，用户决策意图明确，适合优先追踪或生成内容。
- P2：中高价值，场景明确或问题具体。
- P3：普通价值，可作为补充内容。
- P4：低优先级，偏泛。
- P5：暂不建议优先做。

## 6. 推荐内容方向

推荐内容方向使用通用决策场景表达：

- 需求决策指南
- 问题诊断与改善建议
- FAQ 与顾虑解答
- 场景解决方案
- 对比与替代方案
- 信任建立与品牌证明
- 行动前准备清单
- 案例复盘与结果证明

## 7. 如何输入更好的拓词条件

建议在 AI 拓词表单中补充：

- 具体产品、服务、课程、门店、个人品牌或项目方向。
- 目标客户或用户身份。
- 使用场景或咨询场景。
- 不希望生成的候选类型。
- 希望覆盖的决策场景，例如选型、对比、顾虑、场景、行动前准备。

示例约束：

- 不要生成价格词；优先生成家长会问的问题。
- 围绕选型、对比、顾虑和场景展开。
- 面向首次咨询用户，避免承诺效果。

## 8. 为什么候选词不能自动入库

AI 拓词可能产生重复、泛化或不符合项目边界的候选词。为了保证 GEO 提示词资产质量，系统只把拓词结果保存为 `expansion_candidates`，并在页面标记本批重复和库内重复。只有经过人工勾选的候选词，才会保存到提示词策略库。
