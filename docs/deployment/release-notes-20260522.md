## 已进入 main 的主要能力

### 售后问答反馈闭环

- 支持售后问答回答有误反馈。
- 支持反馈待处理列表与反馈详情。
- 支持管理员跟进反馈状态。

### 知识库正式引用规则

- 低可靠资料不进入正式引用范围。
- 待审核资料不进入正式引用范围。
- AI 可引用状态由规则推导，不提供手动开关。

### 知识库列表、目录和管理体验

- 知识库资料列表支持更清晰的卡片 / 表格管理体验。
- 支持知识库目录管理。
- 支持资料移动目录。
- 知识库管理页做了降噪和入口重排。

### 知识库资料规则推荐

- 上传资料和手动录入时，根据文件名、标题和正文前段进行规则推荐。
- 推荐资料类型和资料主题。
- 推荐文案为“系统推荐，可修改”。
- 不调用真实 AI，不自动审核，不自动正式引用。

### 反馈转知识库草稿

- 售后问答反馈可以转换为知识库待审核草稿。
- 转出草稿默认待审核，不自动进入正式引用范围。
- 支持防重复转换和详情跳转。

### 公司用户与部门权限收口

- `company_admin` 可以管理本公司普通用户。
- 支持新增用户、绑定部门、调整 operator / viewer、停用启用、重置密码。
- 防止跨公司管理、越权提权、修改平台管理员和停用自己。

### 模型覆盖记录编辑 / 作废 / 恢复

- 支持编辑模型覆盖记录的结果类字段。
- 核心归属字段保持只读。
- 支持软作废和恢复。
- 作废记录默认不进入列表和报表统计。

### 产品线说明字段

- 产品线支持“产品线说明”。
- 创建、编辑、列表返回和前端展示均已支持。
- 空说明显示为“未填写”。

### 真实 AI 调用安全收口

- Provider 错误脱敏。
- 真实 AI 调用入口增加提示。
- 模型覆盖联网检测增加二次确认。
- operator / viewer 不可触发真实联网检测。
- 设置页不展示 API Key，只展示 Provider 状态说明。

### AI Token 用量统计

- 使用统计页增加 AI 用量区域。
- 支持按公司、用户、部门、功能模块、Provider 统计。
- token unknown 显示为未知，不伪造成真实 0 token。
- 不做价格模型，不做硬性额度限制。

### HelpView warning 清理

- 清理帮助页既有格式 warning。
- Web lint 结果更干净。

### 测试数据清理

- 第一批明显测试公司整包清理已完成。
- 主公司“凯基特”和主知识库保留。
- `operation_logs`、`ai_usage_records`、`ai_call_logs` 保留。
- 主公司下 1 个已停用测试目录已精确清理。
